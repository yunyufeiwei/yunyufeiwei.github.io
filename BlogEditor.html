        使用说明：

        1. 填写文章信息（标题、ID、分类为必填）

        2. 在编辑器中直接书写内容，使用工具栏设置标题、加粗、列表等格式

        3. 工具栏的「代码块」按钮（</> 图标）后面有语言下拉，支持 C++ / C# / HLSL / GLSL / JavaScript / Python / JSON / XML / CSS / Bash 等。选定语言后再粘贴代码，导出后阅读端会自动高亮

        4. 图片可以直接粘贴（Ctrl+V）或从文件管理器拖入编辑器；导出 JSON 时 base64 图片会被替换为占位提示，需手动把图片保存到 assets/ImageWork/Blog/<分类>/ 并把占位文本替换为相对路径

        5. 点击「导出 JSON」会弹出系统保存对话框，请把文件保存到 assets/data/blog/<分类>/<文章ID>.json。保存完成后会询问是否同步更新分类索引（assets/data/blog/<分类>.json），同意后选中索引文件即可自动追加条目

        6. 点击「导出 Markdown」生成 .md 文件内容，可保存到 assets/data/posts/ 目录

        7. 编辑器内容会自动保存到浏览器本地存储，刷新页面不会丢失

        8. 修改已发布文章：点击页面顶部的「加载已有文章」选择对应的 JSON 文件，或者在博客详情页点击「编辑此文」直接载入。修改完成后点「保存修改」会写回原文件，无需再选择保存路径，也不会重复注册到分类索引

        提示：直接选择保存路径需要 Chrome / Edge 等 Chromium 内核浏览器，且通过 https 或 http://localhost 访问页面；其他情况会自动退化为下载文件。
